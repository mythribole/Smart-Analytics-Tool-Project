Smart Analytics Tool
Project Overview
Smart Analytics Tool is an interactive data analysis application built using Streamlit, Pandas, and Plotly.
The application allows users to upload any CSV dataset and automatically perform data exploration, statistical analysis, and visualization.

Features
•	Upload CSV files
•	Dataset Preview
•	Dataset Shape Analysis
•	Column Information
•	Missing Values Analysis
•	Statistical Summary
•	Interactive Data Visualization
•	Histogram Generation
•	Box Plot Generation
•	Bar Chart Generation

Dataset Analysis
The application automatically analyzes the uploaded dataset and provides:
•	First 5 Records
•	Dataset Dimensions
•	Column Names
•	Missing Values Report
•	Statistical Summary

Visualizations
The tool supports the following interactive visualizations:
Histogram
Used to understand the distribution of numerical data.
Box Plot
Used to identify outliers and data spread.
Bar Chart
Used to compare frequency counts of values.

Technologies Used
•	Python
•	Streamlit
•	Pandas
•	Plotly

How to Run
Install Dependencies
pip install -r requirements.txt
Run Application
python -m streamlit run app.py
